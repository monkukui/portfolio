## サイバーエージェント：CA Tech Dojo 育成型インターンシップ, サーバーサイド Go 編
- 期間：2020 夏, 3 週間
- 使用技術：Go, echo, クリーンアーキテクチャ

与えられた api の仕様をもとに、Go で api 開発を行う育成型のインターンシップ。
メンターの社員から手厚いサポートを受け、api 開発のイロハを学びました。

最終日に成果発表をし、「周りに一番影響を与えた賞」をいただきました。
